## Gradient hero
Fundal hero „Ofertă limitată": gradient mov pal → alb (dinspre stânga-sus), tonuri lavandă/mov deschis (#E9DFF5 → #FFFFFF aprox), cu forme organice/blob decorative.

## Tipografie (desktop)
- h1: Playfair Display 700, ~48px
- h2: Playfair Display 700, ~32px
- h3: Playfair Display 700, ~31px
- body/p: Poppins 400, 16px, culoare #051145
- linkuri: Poppins 400, 16px, #1C7261
- buton: Poppins, 16px, text alb #FCFCFC pe fundal #A7408F, colțuri rotunjite

## Butoane
- Primar: fundal magenta `#A7408F`, text alb, border-radius mediu (~6px), padding generos. Ex: „Programează-te", „Trimite programare".
- Secundar/outline apare pe carduri: „Descoperă" (magenta) + „Tarife".

## Layout
- Container max-width ~1140–1200px, centrat.
- Carduri servicii: fundal alb/gri foarte deschis, umbră subtilă, colțuri ușor rotunjite, imagine sus + titlu (Playfair) + descriere (Poppins).
- Grid faciale: 3 coloane (6 carduri). Grid corporale: 3 coloane (9 carduri).
